# Password Generator

## Description
This is a simple Password Generator application developed using Python. The program generates a strong and random password based on the length specified by the user.

## Features
- Generates random passwords
- User-defined password length
- Uses uppercase letters, lowercase letters, numbers, and special characters
- Simple command-line interface

## Technologies Used
- Python 3
- random module
- string module

## How to Run

1. Save the program as `password_generator.py`.
2. Open a terminal or command prompt.
3. Navigate to the project folder.
4. Run the following command:

```bash
python password_generator.py
```

5. Enter the desired password length.
6. The generated password will be displayed on the screen.

## Sample Output

```text
Enter the desired password length: 12
Generated Password: A@7kP#2m!X9q
```

## Code Explanation

- The `random` module is used to select random characters.
- The `string` module provides letters, digits, and special characters.
- The user enters the desired password length.
- The program generates a password by randomly selecting characters and combining them.
- The generated password is displayed to the user.
